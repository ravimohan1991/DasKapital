\section{Definitions}
This section consists of abstractions which will be summoned in the rest of the material.  Please try to be as cooperative as possible with them.

\begin{definition}
  A \emph{commodity} is an abstraction such that
  \begin{itemize}
  \item it is outside \emph{human} context
  \item it results in human satisfaction
  \item represents collection of \emph{some} properties
  \end{itemize} 
\end{definition}
